A. User Experience Requirements

User Stories

  1. As a student with color blindness, I want to be able to customize the color and font on my exam screen, so that I am able to properly comprehend the images and information the text conveys.
  2. As a student with dyslexia, I want to be able to increase the font size and spacing of text questions, so that I am able to navigate the exam at the same level as someone without dyslexia.
  3. As a student with anxiety, I want to view clear feedback and instructions, so that I am assured of my status during the exam, and so that I conduct myself within the expectations of the audio and visual monitoring.
  4. As a student with a visual impairment, I want to be able to customize the contrast and brightness of my screen, so that I do not face any challenges with the visual setup of my examâ€™s content.
  5. As a student with ADHD, I want to be able to customize the layout of test questions, so that I can focus on one question at a time, rather than being distracted by the following question text.
  6. As a student of color, I want access to the camera brightness and contrast settings, so that during the pre-check process I can avoid being flagged or facing any other issues.
  7. As a student with a hearing impairment, I want an icon that allows me to access a text-to-speech tool, so that I can select text and have it read out loud to me.
  8. As a student, I would like to be able to access different screen customizations, so that I can take my exam in a manner efficient to me.
  9. As a student from a non-english background, I want access to an in-built tool that can translate words for me, so that I am able to comprehend words in the exam context if anything becomes confusing for me.
  10. As an educator, I want to be able to adjust my exam and activate/deactivate certain customizable features, so that my exam is taken where no one student has a clear advantage over another.
